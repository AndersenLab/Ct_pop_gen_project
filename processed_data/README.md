## processed_data

This directory contains processed files generated for downstream analyses in the C. tropicalis PopGen manuscript.

### mosdepth_coverage/

Coverage threshold files generated by mosdepth for individual strains.

- `mosdepth_coverage/thresh/*.thresholds.bed`  
  Per-strain coverage threshold files used for downstream coverage-based analyses.

---

### gt_check_only_HDRs/

- `gt_check_only_HDRs/gtcheck_onlyHDR.tsv`  
  Pairwise genetic similarity results calculated using HDR regions only.

---

### gt_check_no_HDRs/

- `gt_check_no_HDRs/gtcheck_noHDR.tsv`  
  Pairwise genetic similarity results calculated excluding HDR regions.

---

### genomic_bins/

- `genomic_bins/ONT_NIC58_1kb_bins.bed`  
  Genomic bin definitions for the NIC58 reference genome in 1 kb windows.

---

### gene_enrichment/

- `gene_enrichment/CT_chromosome_domains.tsv`
  Chromosome domains of NIC58 used for definiing chromosome arms, centers, and tips

- `gene_enrichment/N2_NIC58_orthogroups.tsv`
  OrthoFinder-identified orthologs between *C. elegans* N2 and *C. tropicalis* NIC58

- `gene_enrichment/NIC58_IPR_allApps_20251202.tsv`
  InterProScan annotation results output from `../scripts/gene_enrichment/InterProScan.sh`

- `gene_enrichment/c_elegans.PRJNA13758.WS283.csq.PCfeaturesOnly.longest.gff3`
  Longest isoform protein-coding gene models of *C. elegans* N2 strain  

- `gene_enrichment/c_tropicalis.NIC58_20251002.csq.longest.PC.noMtDNA.fa`
  NIC58 proteome used for InterProScan annotation in `../scripts/gene_enrichment/InterProScan.sh`

- `gene_enrichment/c_tropicalis.NIC58_20251002.csq.longest.gff3`
  Longest isoform protein-coding gene models of *C. tropicalis* NIC58 strain

---

### LD_three_species/

- `LD_three_species/C_tro_all.summary`  
  Linkage disequilibrium summary file for C. tropicalis.

- `LD_three_species/C_bri_all.summary`  
  Linkage disequilibrium summary file for C. briggsae.

- `LD_three_species/C_ele_all.summary`  
  Linkage disequilibrium summary file for C. elegans.

---

### Ct_admixture/

Processed ADMIXTURE output files across multiple K values and replicate seeds.

- `Ct_admixture/LD_0.9_<K>_<seed>.Q`  
  ADMIXTURE ancestry proportion files generated from LD-pruned variants.

- `Ct_admixture/admix_replicates_CV.tsv`  
  Summary table of cross-validation errors across ADMIXTURE replicates.

- `Ct_admixture/seeds.txt`  
  Random seeds used for replicated ADMIXTURE analyses.

---

### Ct_admixture_k28/

Processed ancestry tables for the selected K = 28 analyses.

- `Ct_admixture_k28/K28_Processed_Ancestry_replicate_*.tsv`  
  Processed ancestry tables for replicate runs at K = 28.

- `Ct_admixture_k28/K28_best_k_long_admix_pops_replicate_*.csv`  
  Long-format ancestry tables for replicate runs at K = 28.

- `Ct_admixture_k28/best_k_long_admix_pops_replicate_*.csv`  
  Additional processed long-format ancestry tables associated with K = 28.

---

### Ct_admixture_best_k/

Files used to compare ADMIXTURE results across K values.

- `Ct_admixture_best_k/concat_Qfiles_K*.tsv`  
  Concatenated ancestry proportion tables for each tested K.

- `Ct_admixture_best_k/cv_matrix.tsv`  
  Matrix of cross-validation errors across K values.

---

### LR_calls/

Stats for HDR calls

---

### LD_pruned_trees/

Phylogenetic tree files generated from LD-pruned variants.

- `LD_pruned_trees/LD_0.9/phy_file_LD_0.9.phy.contree`  
  Consensus tree generated from LD-pruned variants.

- `LD_pruned_trees/LD_0.9/phy_file_LD_0.9.phy.contree.rooted`  
  Rooted consensus tree generated from LD-pruned variants.

- `LD_pruned_trees/LD_0.9/phy_file_LD_0.9.phy.log`  
  Log file associated with phylogenetic tree generation.

---

### genome_alignments/

Pairwise genome alignment outputs and transformed coordinate files.

- `genome_alignments/*.delta`  
  NUCmer alignment output files.

- `genome_alignments/*.coords`  
  Coordinate files derived from genome alignments.

- `genome_alignments/*_transformed.tsv`  
  Transformed coordinate tables for downstream analyses.

#### other_species_divergent/

- `genome_alignments/other_species_divergent/*.tsv`  
  Alignment summary files for divergent comparisons involving other species.

---

Geographic metadata, RGs assignments, and saved plotting objects.

- `geo_info/Ct_isotype_list_Hawaii.csv`  
  Hawaii isotype list.

- `geo_info/Ct_isotype_geo_freq.csv`  
  Geographic frequency table of isotypes.

- `geo_info/Ct_lineage_all.csv`  
  RGs assignments for all strains or isotypes.

- `geo_info/Ct_indep_isotype_info_geo.csv`  
  Geographic metadata for independent isotypes.

- `geo_info/geo_and_lineage.csv`  
  Combined geographic and RGs metadata table.

- `geo_info/non_admixed_isotype_replicate_*.txt`  
  Lists of non-admixed isotypes from replicate ADMIXTURE analyses.

- `geo_info/p_isotype_hist.rds`  
  Saved R object for isotype histogram plotting.

- `geo_info/p_strain_hist.rds`  
  Saved R object for strain histogram plotting.

- `geo_info/strain_map.rds`  
  Saved R object for strain map plotting.

- `geo_info/isotype_map.rds`  
  Saved R object for isotype map plotting.
